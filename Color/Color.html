How to change color style?

This feature will be available on the next version of CSS update!

@defaultColor: #000 and @primaryColor: #2F5E92 are 2 base colors of eXo Platform style. The Platform is composed by gray (defaul color) and electric blue (primary color). These colors are the standard ones to generate the other color schemes. For example gray gradient, electric blue gradient, text and graphic element color.

In case you want to change the main color style, simply modify the base color code in the file variables.less in Platform or the correspondent generated css file. In UX Guidelines, it is in exo-platform.css

For example, instead of electric blue for primary color style, users can replace #2F5E92 by #C72222. So, we have@primaryColor: #C72222.
